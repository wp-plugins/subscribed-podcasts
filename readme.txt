Veröffentliche deine abonnierten Podcasts einfach auf deinem Blog.

== Description ==
Exportiere einfach deine abonnierten Podcasts als OPML File und lade diese Datei in dein Wordpress und schon kannst du deine abonnierten Podcasts auf deinem Blog darstellen.

== Installation ==
1. Unzip `subscribed-podcasts.zip`
1. Upload `subscribed-podcasts` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
= Wie bekomme ich meine Podcasts ins Wordpress? =

Exportiere deine abonnierten Podcasts als OPML File.
Lade diese Datei über das Plugin ins Wordpress.
Speicher deine Auswahl und aktualisiere die Feeds.

= Wie kann ich die Podcasts auf meiner Seite darstellen? =

An einer beliebigen Stelle kannst du den Shortcode `[subscribed-podcasts]` einfügen. Dieser generiert dir dann deine Feedübersicht.
Noch ein wenig CSS magie und dann sieht das ganze auch schick aus.

= Warum sieht die Übersicht so schlecht aus? =

Die Übersicht wurde absichtlich nicht gestyled. Das soll ganz dir überlassen werden, damit es deinem Blogdesign entsprechen kann.
